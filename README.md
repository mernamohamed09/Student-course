Student Course Management System 
Description 
This is a C++ console application that manages students and their enrolled courses. The system allows users to add, remove, search, and display students, as well as manage course enrollments and sort students by GPA. 
  
Features 
•	Add a new student (ID, Name, GPA) 
•	Remove a student by ID 
•	Search for a student 
•	Display all students 
•	Enroll a student in a course 
•	Show student courses 
•	Sort students by GPA 
•	Save data to file 
•	Load data from file 
  
Technologies Used 
•	C++ 
•	Object-Oriented Programming (OOP) 
•	STL (vector, algorithm) 
  
 
 
 
 
Project Structure 
Project/ 
│── Student.h 
│── Student.cpp 
│── student course.cpp(main) 
│── students.txt    
  
 
Notes 
•	Each student must have a unique ID 
•	GPA must be between 0.0 and 4.0 
•	Course names should not contain spaces 
•	Duplicate courses are not allowed
